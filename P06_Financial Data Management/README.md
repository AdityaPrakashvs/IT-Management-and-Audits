# 💰 Financial Data Manager (P06)

This project focuses on designing, managing, and analyzing financial datasets using SQL and Python. It simulates real-world financial data pipelines with data quality checks and reporting.

---

## 🎯 Objective

* Design relational financial databases
* Generate realistic financial datasets
* Perform advanced SQL analytics
* Ensure data quality and compliance

---

## 🧠 Key Concepts

* Relational Database Design
* SQL (JOIN, CTEs, Window Functions)
* Data Quality Frameworks
* Financial Data Analytics

---

## ⚙️ Tools & Technologies

* Python (SQLite3, Faker)
* SQL
* Click (CLI)
* Rich (Terminal UI)

## 🚀 Features

### 🔹 Database Design

* Schema with tables, keys, and relationships
* Supports banking & fintech datasets

### 🔹 Data Generation

* Synthetic financial data using Faker
* Realistic transactions (UPI, GST, etc.)

### 🔹 SQL Analytics

* Advanced queries:

  * Window functions
  * Aggregations
  * Subqueries

### 🔹 Data Quality Checks

* Accuracy
* Completeness
* Consistency
* Validity

👉 Critical for RBI compliance in financial systems
