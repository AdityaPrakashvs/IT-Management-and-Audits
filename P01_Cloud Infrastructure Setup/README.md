# ☁️ P01 – Lab Setup & IT Infrastructure (AWS EC2)

This project contains a detailed lab walkthrough for setting up cloud infrastructure using AWS. It covers deploying a virtual machine, configuring security, and hosting a web server.

---

## 🎯 Objective

* Understand cloud infrastructure basics
* Deploy and configure an AWS EC2 instance
* Set up secure remote access
* Install and run a web server
* Monitor cloud resources and usage

---

## 🧠 Key Concepts

* Cloud Computing (IaaS)
* AWS EC2 & Virtual Machines
* Security Groups & Networking
* SSH Access
* Web Server Deployment (Nginx)
* Cloud Monitoring (CloudWatch)

---

## ⚙️ Tools & Technologies

* AWS EC2
* AWS CloudWatch
* Ubuntu (Linux)
* Nginx
* Terminal / SSH

---

## 📂 Project Files

## 🚀 What This Lab Covers

* Creating AWS Free Tier account
* Launching EC2 instance
* Configuring security groups (ports 22, 80, 443)
* Connecting via SSH
* Installing and running Nginx server
* Monitoring system performance
