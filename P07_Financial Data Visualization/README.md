# 📊 Financial Data Visualizer (P07)

This project focuses on transforming financial data into meaningful visual insights using Python and interactive dashboards.

---

## 🎯 Objective

* Analyze financial datasets
* Create multiple chart types
* Build interactive dashboards

---

## 🧠 Key Concepts

* Data Visualization Principles
* Statistical Analysis
* Dashboard Design

---

## ⚙️ Tools & Technologies

* pandas
* matplotlib
* plotly
* seaborn

## 📂 Project Files

## 🚀 Features

### 🔹 Data Analysis

* Moving averages
* Growth rates (YoY)
* Percentiles

### 🔹 Visualization Types

* Line charts (trends)
* Bar charts (comparisons)
* Pie charts (distribution)
* Heatmaps (patterns)
* Candlestick charts

### 🔹 Dashboard

* Interactive HTML dashboard
* Filters and drill-down

---

## 📊 Insights

* Visualization converts data into decisions
* Proper chart selection is critical
* Avoid misleading visuals (chartjunk)
