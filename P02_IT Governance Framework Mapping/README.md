# 🏛️ IT Governance Framework Mapper (P02)

This project demonstrates how to map organizational IT processes to industry-standard governance frameworks like **COBIT 2019** and **ITIL v4** using a Python-based tool.

---

## 🎯 Objective

* Understand IT governance frameworks (COBIT & ITIL)
* Map organizational IT processes to governance objectives
* Perform gap analysis to identify missing controls
* Generate compliance scorecards and reports

---

## 🧠 Key Concepts

* IT Governance vs IT Management
* COBIT 2019 (5 domains, 40 objectives)
* ITIL v4 (3 categories, 34 practices)
* Process Mapping & Keyword Matching
* Gap Analysis
* Compliance Reporting

---

## ⚙️ Tools & Technologies

* Python (3.8+)
* CLI-based Governance Mapping Tool
* JSON (for process definition)
* Git & Terminal
* HTML Reporting

## 🚀 What This Project Covers

### 🔹 1. Environment Setup

* Clone governance mapping tool repository
* Set up Python virtual environment
* Install dependencies and run CLI

### 🔹 2. Framework Exploration

* Analyze **COBIT 2019 domains**:

  * EDM (Governance)
  * APO (Planning)
  * BAI (Implementation)
  * DSS (Operations)
  * MEA (Monitoring)

* Analyze **ITIL v4 practices**:

  * General Management
  * Service Management
  * Technical Management

---

### 🔹 3. Process Mapping

* Map IT processes to COBIT objectives
* Map IT processes to ITIL practices
* Evaluate mapping using confidence scores

---

### 🔹 4. Gap Analysis

* Identify uncovered governance areas
* Measure framework coverage (%)
* Detect governance risks and blind spots

---

### 🔹 5. Report Generation

* Generate terminal-based reports
* Generate HTML compliance reports
* Review governance scorecards

---

### 🔹 6. Custom Process Mapping

* Create your own IT processes (JSON format)
* Map them to frameworks
* Generate personalized governance reports

---

## 📊 Sample Output

* COBIT Coverage: ~60–65%
* ITIL Coverage: ~55–60%
* Governance Scorecard generated
* HTML report with domain-wise insights

## 🌍 Real-World Application

This project is highly relevant for:

* IT Auditors
* Risk & Compliance Analysts
* FinTech Companies
* Banking & Financial Institutions

Frameworks like COBIT & ITIL are widely used by organizations to meet compliance requirements from regulators like **RBI** and **SEBI**.

---

## 💡 Key Learning

* COBIT defines **what should be achieved (governance)**
* ITIL defines **how to deliver IT services (operations)**
* Combining both provides **complete IT governance coverage**
